About

I am Valencia Lie, currently a Y1 Undergraduate Student pursuing Data Science and Artificial Intelligence in Nanyang Technological University, Singapore. This website serves as a place for me to showcase the projects I have done, both in school and during my free time. Feel free to contact me if you have any enquiries.

Portfolio

  • K-Drama Chemistry Matcher Data Science Project

    • A project that my groupmates and I created for our CZ1016 Introduction to Data Science module. The project aims to match Korean actors and actresses based on their perceived chemistry, mainly through keyword-specific sentiment analysis. The project also utilises facial similarity to tackle the cold start problem of newcomers by recommending a similar looking actor/actress' best match. A 3-month long project but it is probably the most rewarding project out of all because I learnt so much and it is so fun doing a project that I have an interest in!

    Click here for codes. Click here for visualisation dashboard.

  • NTU CAmpcOde Hackathon with Roboto Coding Academy

    • A project that my groupmates and I did for the hackathon. The first challenge requires us to train a model to analyse words and sentences in chat communication to predict the mood of the conversation. The second challenge requires us to train a model to analyse images to predict emotions. The third challenge requires us to develop a conceptual solution to address the statement of "How can team collaborating remotely be more productive than in a face-to-face meeting?". This is my very first hackathon and also the first hackathon where I clinched an award (1st Runner Up)!

    Click here for Challenge 1 and 2 codes. Click here for powerpoint slide on Challenge 3.

  • MLH Hackathon: Innovation in HealthTech

    • A project that my groupmates and I did for the hackathon. The hackathon requires us to create something innovative that can revolutionise the Health Industry. The project my groupmates and I did was using Data Science and Machine Learning to help detect a disease based on a person's symptoms. Although we only ended up doing 2 diseases (cervical cancer and COVID) because of the lack of data and the dashboard that we did was unfortunately not completed, it was nevertheless a very fruitful experience as it is my very first 48 hour hackathon.

    Click here for codes.

  • Machine Learning Capstone Project with Algoritma Data Science School
  • Time Series mini project

    • I did 2 projects that mainly uses time series models. The first project is a project that I did for my Machine Learning Capstone Project. This project mainly deals with time series analysis where I used and compared the results of different time series analysis tools like Simple Moving Average, Simple, Double and Triple Smoothing Exponential Smoothing, Seasonal AutoRegressive Integrated Moving Average and STLM. This project aims to predict the number of visitors of an F&B according to the date. The second project is also very similar to the first one but it aims to forecast future data about New Delhi's mean temperature based on past data.

    Click here for Capstone Project's codes. Click here for mini project's codes.
