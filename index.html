  • Singapore Airlines App Challenge (Future of Work) - Work and Leisure Dashboard

    • A project that I did for another airline-related hackathon! This time, I was tasked to revolutionise the way employees work remotely in order to increase productivity and improve spontaneity in collaboration. Although it seems counter-intuitive, I have come up with the conclusion that the only way employees will be working productively is if they have sufficient breaks and so I devise a way so that employees can work and have fun. I incorporated a multi-player game made with Unity into a R Shiny Dashboard which allows employees to access the game after certain amount of hours of working. I also designed a reward system within the game to incentivise employees to use it to interact with their colleagues, which hopefully can spark some spontaneous ideas and collaboration. Although I did not win this, I feel like this is a very fun yet challenging project and I'm happy I did it.

  • Introduction to Data Science Project - Korean Drama Actor Chemistry Matcher

    • A project that my groupmates and I created for our CZ1016 Introduction to Data Science module. The project aims to match Korean actors and actresses based on their perceived chemistry, mainly through keyword-specific sentiment analysis. The project also utilises facial similarity to tackle the cold start problem of newcomers by recommending a similar looking actor/actress' best match. A 3-month long project but it is probably the most rewarding project out of all because I learnt so much and it is so fun doing a project that I have an interest in! This project also achieved very outstanding response from our Professor because he awarded us the highest score (76/80) in the whole cohort!
